IMPLEMENTATION SUMMARY
======================
- PingCodeClient: constructor accepts `grant_type` (default 'client_credentials')
- loadCachedToken: returns {grant_type, access_token, refresh_token, expires_at} object; backward-compatible with old format
- saveCachedToken: writes {grant_type, access_token, refresh_token?, expires_at}; backward-compatible 3-arg calls
- readRawTokenCache: new helper reads cache file regardless of expiry
- accessToken(): handles both grant types; authorization_code with expired access_token uses refresh_token
- exchangeAuthorizationCode(code, redirectUri): exchanges auth code for tokens, caches result
- refreshAccessToken(refreshToken): refreshes access token, preserves existing refresh_token
- buildAuthorizationUrl(redirectUri, state): builds /oauth2/authorize URL
